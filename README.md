## 🇧🇷 Português do Brasil

Uma aplicação web projetada para criar paisagens sonoras personalizadas. Melhore seu foco durante o desenvolvimento, relaxe após um longo dia ou crie o ambiente perfeito para dormir combinando diferentes sons simultaneamente.

### ✨ Funcionalidades
* **Mixagem de Áudio Simultânea:** Reproduza múltiplos sons (chuva, lareira, cafeteria, etc.) ao mesmo tempo.
* **Controle de Volume Dinâmico:** Ajuste o volume global ou calibre o volume individual de cada faixa de áudio para a mistura perfeita.
* **Controles Mestres:** Botões rápidos para "Tocar Tudo" ou "Parar Tudo".
* **Interface Moderna:** Design responsivo utilizando a estética *Glassmorphism* com suporte nativo a Tema Escuro (Dark Mode).

### 🛠️ Tecnologias Utilizadas
A aplicação foi construída com foco em performance e simplicidade, utilizando as capacidades nativas do navegador moderno:
* **HTML5:** Estrutura semântica e acessibilidade.
* **CSS3:** Variáveis nativas (Custom Properties), Flexbox, CSS Grid e animações suaves.
* **JavaScript (Vanilla):** Manipulação de DOM, HTML5 Audio API e Promises.

### 🚀 Como executar o projeto
Para utilizar a aplicação, não é necessária nenhuma instalação prévia. Basta acessar o link do projeto hospedado:
[https://johnwiliam.github.io/SonsRelaxantes/](https://johnwiliam.github.io/SonsRelaxantes/)

## 🇺🇸 English

A web application designed to create personalized soundscapes. Improve your focus during development, relax after a long day, or create the perfect environment for sleep by combining different sounds simultaneously.

### ✨ Features
* **Simultaneous Audio Mixing:** Play multiple sounds (rain, fireplace, coffee shop, etc.) at the same time.
* **Dynamic Volume Control:** Adjust the global volume or calibrate the individual volume of each audio track for the perfect mix.
* **Master Controls:** Quick buttons to "Play All" or "Stop All".
* **Modern Interface:** Responsive design using *Glassmorphism* aesthetics with native Dark Mode support.

### 🛠️ Technologies Used
The application was built with a focus on performance and simplicity, using the native capabilities of modern browsers:
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Native variables (Custom Properties), Flexbox, CSS Grid, and smooth animations.
* **JavaScript (Vanilla):** DOM manipulation, HTML5 Audio API, and Promises.

### 🚀 How to run the project
To use the application, no installation is required. Simply access the hosted project link: 
[https://johnwiliam.github.io/SonsRelaxantes/](https://johnwiliam.github.io/SonsRelaxantes/)
