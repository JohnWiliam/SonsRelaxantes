## 🇧🇷 Português do Brasil

Uma aplicação web projetada para criar paisagens sonoras personalizadas. Melhore seu foco durante o desenvolvimento, relaxe após um longo dia ou crie o ambiente perfeito para dormir combinando diferentes sons simultaneamente.

### ✨ Funcionalidades
* **Mixagem de Áudio Simultânea:** Reproduza múltiplos sons (chuva, lareira, cafeteria, etc.) ao mesmo tempo.
* **Controle de Volume Dinâmico:** Ajuste o volume global ou calibre o volume individual de cada faixa de áudio para a mistura perfeita.
* **Controles Mestres:** Botões rápidos para "Tocar Tudo" ou "Parar Tudo".
* **Interface Moderna:** Design responsivo utilizando a estética *Glassmorphism* com suporte nativo a Tema Escuro (Dark Mode).

### 🛠️ Tecnologias Utilizadas
A aplicação foi construída com foco em performance e simplicidade, utilizando as capacidades nativas do navegador moderno:
* **HTML5:** Estrutura semântica e acessibilidade.
* **CSS3:** Variáveis nativas (Custom Properties), Flexbox, CSS Grid e animações suaves.
* **JavaScript (Vanilla):** Manipulação de DOM, HTML5 Audio API e Promises.
